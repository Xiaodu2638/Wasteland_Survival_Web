# 桌游：废土求生 游玩手册

### 一、游玩流程

1．所有玩家随机从角色卡中各抽取一名角色；

2．所有玩家中决定回合顺序，并将（含陷阱）物资卡混洗成一沓放在桌面上；

3．第一名玩家首先从随机事件卡中抽取一张，作为本回合的随机事件；

4．玩家按顺序在自己的回合分别在物资卡顶部抽取一张手牌暗纳入自己的手牌中（若为陷阱卡则将其正面朝上放置于物资卡一侧，该玩家立即结算该陷阱卡的负面效果并换到下家回合）；

5．每三个陷阱被触发时，将陷阱卡洗入牌堆中，重置陷阱概率；

6．依此类推直到决出最后胜利的玩家；

### 二、获胜条件：除胜利者之外的所有玩家皆出局

发生以下情况时候，玩家会判定出局：

1.  **生命垂危**：健康归零；
2.  **器官衰竭**：饱食和水分同时归零；
3.  **自我终结**：在拥有武器的情况下理智归零，角色自杀出局；
4.  **全身溃烂**：辐射达到上限，角色立即死亡出局；

### 三、卡牌介绍

1．**角色卡**：游戏中角色包含健康、饱食、水分、辐射、理智五类基础属性，不同的角色用用不同的属性和特殊机制，玩家们需要根据选到的角色制定游戏方案（所有玩家辐射上限都为5点，其余属性初始值为上限值）；

① **流浪者**：凭着在街头的流浪经历，靠着一块黑面包就就能活上一天

初始属性：健康5点；饱食4点；水分4点；辐射0点；理智5点；

特性：饥饿专家（食物、饮品的效果翻倍）；

画饼充饥（可以消耗“空罐头”增加一点饱食）

② **伐木工**：凭借着在深林中工作的经验，总是能在危险面前全身而退

初始属性：健康5点；饱食5点；水分5点；辐射0点；理智5点；

特性：从容不迫（在遇到陷阱卡时，可选择弃置一张手牌（放到物资卡底部）抵消陷阱效果）；

③ **小偷**：最没有道德边界的人，对偷东西不会感到愧疚

**初始属性**：健康5点；饱食5点；水分5点；辐射0点；理智4点；

**特性**：本性难移（使用行动卡“道德败坏”时不会受到扣除理智的负面效果）；

④ **病人**：虚弱到不行，没人把你放在眼里

**初始属性**：健康4点；饱食4点；水分3点；辐射0点；理智4点；

**特性**：小透明（没人愿意攻击你，你不能被装备武器主动攻击）；

药物专家（药品效果增加一点，且使用抗辐射药物时同时回复一点健康）；

禁忌（酒精会加剧病情，你不能持有物资卡“伏特加“，搜索到后立即丢弃）；

⑤ **退伍军人**：体质优秀，但患有创伤性应激障碍

**初始属性**：健康5点；饱食5点；水分5点；辐射0点；理智3点；

**特性**：战争后遗症（无数战友在你面前死去，你更容易发疯）

正义感（使用行动卡“道德败坏“时，扣除理智2点）

⑥ **雇佣兵**：真正的冷血的杀手，不会向世人流露任何感情

**初始属性**：健康5点；饱食4点；水分4点；辐射0点；理智4点

**特性**：战争狂人（使用行动卡“攻击欲望”时，不会受到扣除理智的负面影响）；

嗜血狂魔（可以弃置两张手牌，充当行动卡“攻击欲望“使用，但会扣除理智2点）；

失败过敏（不能接受自己的失败，在没有武器时理智归零会选择上吊自杀出局）；

2．**行动卡**：

① 攻击欲望：在自己的回合打出，指定一张装备武器对指定玩家造成伤害，使用后理智扣除1点；

② 道德败坏：在其他人的回合打出，随机抽取该回合的玩家一张手牌（如抽取到行动卡需要归还，视为一无所获），使用后理智扣除1点；

③ 合作共赢：在任意回合同一张物资卡打出，剩余玩家需展示决定的交换物，交换对象由出牌方决定，使用后双方理智各增加1点；

3．**物资卡**：

物资卡主要分为消耗品和战备物资：

1.  **消耗品**：食物、饮品、药品（抗辐射药物、普通药物、抗抑郁药物）、无用物资：

**食物类（用于恢复饱食）**

1.  **黑面包：** 恢复1点饱食。
2.  **罐头肉：** 恢复2点饱食。
3.  **生土豆：** 恢复1点饱食，但若当前饱食 ≥4，则无效果（吃不下）。
4.  **发霉饼干：** 恢复1点饱食，获得1点辐射（食物已变质）。
5.  **野果：** 恢复1点饱食，随机失去1点水分或1点理智（可能有毒）。

**饮品类（用于恢复水分）**

1.  **雨水：** 恢复1点水分。
2.  **净水：** 恢复2点水分。
3.  **脏水：** 恢复1点水分，获得1点辐射。
4.  **伏特加：** 恢复1点水分，理智+1，短暂麻痹(止痛)。
5.  **冷却剂（工业用）：** 恢复2点水分，健康-1

**药物类**

**▶ 普通药物（恢复健康）**

1.  **绷带：** 恢复1点健康。
2.  **抗生素：** 恢复2点健康。
3.  **止痛片：** 恢复1点健康，本回合内免疫一次陷阱造成的健康损失（仅限本次回合）。

**▶ 抗抑郁药（恢复理智）**

1.  **镇静剂：** 恢复1点理智。
2.  **抗抑郁药：** 恢复2点理智。
3.  **安慰剂：** 恢复1点理智，但若你当前理智 ≥5，则无效果（心理依赖失效）。

**▶ 抗辐射药（降低辐射）**

1.  **碘片：** 辐射-1，无副作用。
2.  **螯合剂：** 辐射-2，健康-1（强力排毒，身体受损）。
3.  **民间偏方：** 辐射-1，随机失去1点饱食或1点水分（取决于数值高的一项）。

病人角色使用任何抗辐射药时，额外恢复1点健康（按你原设定保留）。

**🗑️无用物资（大部分无用，可以被弃置）**

1.  **破布：** 无效果。可用于“合作共赢”交易，但不能单独使用。
2.  **空罐头：** 无效果。
3.  **旧报纸：** 使用后理智+1（读到希望故事）。
4.  **生锈钥匙：** 无效果。
5.  **玩具熊：** 若你在理智 ≤2 时使用，理智+1（精神寄托）。

1.  **战备物资**：

- 装备武器（配合行动卡“攻击欲望“打出）：

栓动步枪：造成1点伤害；

双管霰弹枪：造成2点伤害；

- **防身武器**（在其他人对你使用行动卡“道德败坏“时打出，阻止偷牌，且对偷拍者造成伤害；在其他人使用行动卡”合作共赢“交易时，打出该牌，阻止一场交易，并使交易双方理智各扣除一点）：

手枪：造成1点伤害；

截断霰弹枪：造成2点伤害；

匕首：抵消伤害并可窃取对方一张手牌；

1.  **陷阱卡**：

- **辐射尘：** 你受到1点辐射。若你当前辐射 ≥3，再失去1点理智。
- **干裂嘴唇：** 你水分减少1点。若水分已为0，则健康减少1点。
- **胃部绞痛：** 你饱食减少1点。若饱食已为0，则健康减少1点。
- **噩梦闪回：** 你理智减少1点。若你当前理智 ≤2，再失去1点水分（彻夜未眠）。
- **野狗尾随：** 你必须弃置一张食物卡，否则健康减少1点。若无食物卡可弃，饱食减少1点。
- **幻觉低语：** 弃置一张手牌。若无手牌，则理智减少1点。
- **污水入喉：** 你水分减少1点，且获得1点辐射。若你当前水分 ≤1，额外失去1点理智。
- **断骨之痛：** 你健康减少1点。若你当前健康 ≤3，则本回合无法使用食物或饮品卡。
- **绝望低语：** 若你当前持有武器，失去1点理智（枪声唤起回忆）。

1.  **随机事件卡**：
2.  **酸雨飘落：** 所有玩家获得1点辐射。若某玩家当前辐射 ≥4，则其额外失去1点理智；
3.  **烈日灼烧：** 所有玩家水分减少1点。若水分已为0，则健康减少1点；
4.  **寒夜刺骨：** 所有玩家饱食减少1点。若饱食已为0，则健康减少1点；
5.  **人心惶惶：** 所有玩家理智减少1点。若某玩家当前理智 ≤2，则其额外失去1点水分；
6.  **废墟搜刮：** 所有玩家暗置摸一张牌；
7.  **净水残留：** 所有玩家水分增加1点（最多补至上限）；
8.  **罐头余粮：** 所有玩家饱食增加1点（最多补至上限）；
9.  **短暂和平：** 本回合内，禁止使用“攻击欲望”和“道德败坏”行动卡；
10. **辐射云掠过：** 所有玩家获得1点辐射；
11. **微弱希望：** 所有玩家理智增加1点（最多补至上限）；
12. **军方介入**持有武器的玩家需立刻丢弃武器，所有玩家健康增加一点；
